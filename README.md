# The Boys Golf App - Version 7.2 Full Prototype Migration

This build restores the full prototype UI inside a Vite project so it can be tested with `npm run dev`.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown by Vite, usually http://localhost:5173.

## Notes

- This version brings the full prototype navigation and workflow back into the production project shell.
- Data is still stored locally in the browser for testing.
- The next production step is connecting these screens to Supabase authentication and database tables.
